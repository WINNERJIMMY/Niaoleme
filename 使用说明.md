# 使用说明

## 推荐方式：使用手机扫码

1. 在手机上安装 **Expo Go** 应用
   - Android: Google Play 或各大应用商店搜索 "Expo Go"
   - iOS: App Store 搜索 "Expo Go"

2. 确保手机和电脑在同一WiFi网络

3. 在终端运行：
```bash
npm start
```

4. 用 Expo Go 扫描终端显示的二维码

## 应用功能

- **首页**: 查看今日排尿统计和广告
- **记录**: 记录尿量、颜色等信息
- **地图**: 查找附近公共厕所
- **分析**: 查看健康分析报告

现在直接扫描终端的二维码即可使用！
