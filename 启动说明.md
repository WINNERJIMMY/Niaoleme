# 安卓应用启动说明

## 环境要求
- Node.js 16+
- Expo CLI
- Android Studio (可选)
- 安卓设备或模拟器

## 安装步骤

1. 安装依赖
```bash
cd frontend
npm install
```

2. 安装Expo CLI (如果没有)
```bash
npm install -g @expo/cli
```

3. 启动开发服务器
```bash
npx expo start
```

4. 在手机上安装Expo Go应用，扫描二维码运行

## 或者构建APK

1. 构建安卓应用
```bash
npx expo build:android
```

2. 下载生成的APK文件安装到手机

## 功能说明

### 首页
- 显示今日排尿统计
- 快捷操作按钮
- 广告展示

### 记录页面
- 记录尿量、颜色
- 添加备注信息
- 自动记录时间

### 地图页面
- 显示附近厕所位置
- 添加新厕所标记
- 查看厕所评价

### 分析页面
- 7天趋势图表
- AI健康分析
- 个性化建议

## 注意事项
- 需要开启位置权限才能使用地图功能
- 首次使用需要联网获取数据
- 建议在真机上测试完整功能
