# 💧 尿了么 (Urine Health Manager) - 开发文档

## 1. 项目概况
*   **项目名称**：尿了么
*   **Slogan**：您的贴身嘘嘘健康小管家
*   **当前版本**：v1.0.0
*   **适用平台**：Android (兼容 Android 4.1+ / SDK 16+)

## 2. 创立目的 (Project Mission)
在快节奏的现代生活中，人们常常忽视身体发出的最直接健康信号——尿液颜色。脱水、肾脏负担往往在不知不觉中发生。传统的医疗健康软件界面冰冷、专业术语晦涩，让用户产生抵触心理。

**“尿了么”** 的诞生旨在解决以下痛点：
*   **健康焦虑**：通过软萌的交互，降低用户记录身体数据的心理门槛。
*   **如厕难题**：在陌生的城市（如两江新区），快速找到干净、有纸的厕所。
*   **数据价值**：通过长期记录和 AI 分析，为用户提供个性化的饮水和健康建议。

## 3. 核心功能 (Core Features)

### 🏠 首页 (Home)
*   **暖心问候**：根据时间段发送喝水提醒。
*   **健康看板**：直观展示今日打卡状态。
*   **商业闭环**：中心预留非打扰式广告位，可对接矿泉水或健康险广告。

### 📝 轻松记录 (Record)
*   **Emoji 色卡**：摒弃复杂的医学参数，使用 💧(透明)、🍋(浅黄)、☕(棕色) 等 Emoji 快速选择尿液颜色。
*   **智能定位**：一键调用 GPS 获取当前“嘘嘘地点”，生成个人的健康地理足迹。

### 🗺️ 找厕所 (Map)
*   **地图导航**：直观展示周边的公厕位置（演示版内置两江新区地图）。
*   **互助评价**：引入社区力量，查看厕所“是否有纸”、“是否排队”、“卫生状况”，解决找厕所像“开盲盒”的尴尬。

### 🤖 智能分析 (Analysis)
*   **对话式交互**：采用 Chatbot 形式，像聊天一样获取健康报告。
*   **多维分析**：
    *   **周报**：统计一周的水分摄入情况。
    *   **AI 深度诊断**：接入大模型（DeepSeek/豆包等）API，针对异常数据给出专业且温暖的医疗建议。

## 4. 技术架构 (Technology Stack)
*   **前端框架**：React Native / Expo
*   **开发语言**：JavaScript / Java (Android Native)
*   **构建工具**：Gradle 8.2 + AGP 7.4.2
*   **兼容性优化**：
    *   开启 `MultiDex` 支持。
    *   `minSdkVersion` 降级至 16，确保在 Android 4.1 老旧设备上也能流畅运行，覆盖下沉市场。
*   **依赖库**：
    *   `expo-location`: 地理位置服务
    *   `react-navigation`: 路由管理
    *   `react-native-maps` (规划中): 实时地图支持

## 5. 未来规划 (Roadmap)
*   **V1.1**: 接入真实后端 API，实现用户数据云端同步。
*   **V1.2**: 接入智能马桶硬件，实现“无感记录”。
*   **V2.0**: 建立尿液健康大数据库，与体检机构合作。
