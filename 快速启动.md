# 快速启动指南

## 1. 安装依赖
```bash
npm install
```

## 2. 启动应用
```bash
npx expo start
```

## 3. 运行方式

### 方式A: 使用Expo Go (推荐)
1. 在手机上安装 Expo Go 应用
2. 扫描终端显示的二维码
3. 应用会自动加载

### 方式B: 使用Android模拟器
```bash
npx expo start --android
```

### 方式C: 构建APK
```bash
npx expo build:android
```

## 常见问题

### 如果遇到错误
1. 删除 node_modules 和 package-lock.json
2. 重新运行 npm install
3. 清除缓存: npx expo start -c

### 如果API连接失败
确保后端服务已启动在 http://localhost:3000
